# 🚀 초진마스터V2 (CHOJINMASTER V2)

초진환자 유입 분석 및 자별 내원횟수 계산을 위한 통합 관리 대시보드입니다.

## 주요 기능
- **통합 대시보드 (Overview)**: 환자 유공 현황, 월별 유입 추이, 내원경로 분석 시각화.
- **환자 현황 관리**: 건보/기타, 자보 환자 목록 관리 및 실시간 상태 업데이트.
- **자보 내원 계산기**: 사고일 및 내원일 기준 진료 기준(첩약, 추나, 약침 등) 자동 계산.
- **월별 통계 레포트**: 누적 환자 분석 및 재진율, 동의율, 탈락사유 분포 파악.
- **데이터 영속성**: 브라우저 `localStorage`를 사용하여 기기 로컬에 안전하게 데이터 보관.
- **엑셀 내보내기**: 전체 환자 데이터 및 통계 내용을 `.xlsx` 형식으로 다운로드.

## 배포 방법 (GitHub Pages)
이 저장소는 HTML 단일 파일로 구성되어 있어 GitHub Pages를 통해 간편하게 배포할 수 있습니다.
1. 이 레포지토리를 GitHub에 업로드합니다.
2. `Settings` -> `Pages`로 이동합니다.
3. `Build and deployment` -> `Branch`에서 `main` 브랜치의 root를 선택하고 Save를 누릅니다.
4. 몇 분 후 제공되는 URL을 통해 대시보드에 접속할 수 있습니다.
